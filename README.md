# Smart Shopping Cart System

A functional desktop application developed with **Python** and **PyQt5**. It allows users to manage a virtual shopping list with automated discounts.



###  Key Features

* **Item Management:** Add items with specific quantities and prices.
* **Special Offers:** 5% discount for *coffee, apple, cereal, cheese, and pasta*.
* **Bulk Discount:** 10% discount for items with quantity > 3.
* **Coupon System:** Supports `WELCOME10`, `SAVE20`, and `MEGA25`.

---

###  How to Run

1. **Install PyQt5**
2. **Run the app**
